# DApp Simple Blog 

A Simple Blog DApp on Ethereum blockchain using Truffle, Solidity and React.js.

### Prerequisites

You need to have installed Node.js, Truffle and set Ropsten network on meta mask.

### Installing
* cd client
* npm install

### To Run
* truffle develop
* > compile
* > migrate
* cd client
* npm start

Code is Running on 
+ http://localhost:3000/
